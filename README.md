
# XML Scraper - Gerador de Catálogo de Alta Performance

Este projeto é uma ferramenta de automação desenvolvida em **Electron** e **Node.js** para extração de dados de catálogos de produtos e imóveis, convertendo-os automaticamente para o formato **XML (RSS/Google Merchant Center)**.

## 🚀 Objetivo
O objetivo principal é eliminar o trabalho braçal de atualização de catálogos para campanhas de tráfego pago, garantindo uma fonte de dados estável, limpa e escalável.

## 🛠 Tecnologias Utilizadas
- **Electron**: Para interface desktop multiplataforma.
- **Node.js**: Motor de execução *backend*.
- **Axios & Cheerio**: Para extração eficiente via requisições HTTP e manipulação de DOM.
- **BrowserWindow API**: Para renderização de sites que utilizam carregamento dinâmico (JavaScript/SPA).

## 📋 Como utilizar
1. Certifique-se de ter o [Node.js](https://nodejs.org/) instalado.
